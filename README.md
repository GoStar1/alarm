# SMTP邮件发送器 Chrome插件

这是一个Chrome浏览器插件，可以通过SMTP协议发送邮件。

## 文件结构

```
alarm/
├── manifest.json    # 插件配置文件
├── popup.html       # 弹出页面
├── popup.js         # 邮件发送逻辑
└── README.md        # 说明文档
```

## 安装方法

1. 打开Chrome浏览器
2. 进入 `chrome://extensions/`
3. 开启右上角的"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择这个文件夹

## 使用方法

1. 点击浏览器工具栏中的插件图标
2. 填写以下信息：
   - SMTP服务器（如：smtp.gmail.com）
   - SMTP端口（如：465）
   - 用户名（你的邮箱地址）
   - 密码/授权码（注意：部分邮箱需要使用授权码而非登录密码）
   - 收件人
   - 邮件主题
   - 邮件内容
3. 点击"发送邮件"按钮

## 常用邮箱SMTP配置

### Gmail
- SMTP服务器: smtp.gmail.com
- 端口: 465 (SSL) 或 587 (TLS)
- 需要开启"两步验证"并使用"应用专用密码"

### QQ邮箱
- SMTP服务器: smtp.qq.com
- 端口: 465
- 需要在邮箱设置中开启SMTP服务并获取授权码

### 163邮箱
- SMTP服务器: smtp.163.com
- 端口: 465
- 需要在邮箱设置中开启SMTP服务并获取授权码

## 注意事项

- 插件配置会自动保存到本地存储
- 发送邮件需要网络连接
- 请勿将密码分享给他人
